## Testing

Using [jest](https://facebook.github.io/jest/) and [sass-true](http://oddbird.net/true/) run

```
$ npm test
```

Inside each mixin folder there is a `[name].test.scss` and `[name].test.js`

See Sass-True library docs for more info
